小小便利贴 Desktop · Python · Tkinter

这不是网页小程序，而是可以在电脑桌面上弹出暖心便签的 Python 小工具。GitHub Pages 只负责托管这个说明页面， 真正的程序需要在本地运行。

如何运行「暖心便利贴」程序？

仓库中目前有两个主程序文件：

  • love_notes.py – 新版暖心便签，更多优化和文案
  • sticky_notes.py – 初版便利贴效果

在 Windows 上的推荐使用方式：

  1. 先安装 Python 3.10+ （从 python.org 安装）。
  2. 把这个仓库克隆或下载到本地，例如桌面某个文件夹。
  3. 在该文件夹空白处按住 Shift 并右键 → 选择“在终端中打开”。
  4. 在终端中输入：python love_notes.py 然后回车。

如果系统里有多个 Python 版本，可以尝试 python3 或 py 命令。

小提示 & 自定义文案

程序内置了一些中文暖心句子，你也可以自己准备一份 .txt 文件，每一行是一条小句子，然后在终端中使用：

python love_notes.py --messages 我的文案.txt

这样每次弹出的便利贴，就会随机从你提供的文案中选择一条。
